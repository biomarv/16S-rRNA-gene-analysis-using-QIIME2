# Qiime2 pipeline MetaBac Lab

## Set the environment 

```{bash}
conda activate qiime2-2021.2
```

## Prepare your files

Create project folder and change into it

```{bash}
mkdir PROJECT_NAME
cd PROJECT_NAME
```

Assign your metadata file to a variable

```{bash}
METADATA="/home/user/PROJECT_NAME/metadata-file.tsv"
```
